ems蓝牙协议

前提

该文档定义了ems智能设备与手机之间的蓝牙协议，ems设备采用双蓝牙模块，均工作在蓝牙从模式。协议载荷部分采用json格式，考虑嵌入式设备的运算、内存等因素，json内不得出现中文。
| 简称 | 定义 |
|–| – |
| 蓝牙a | 与用户手机连接的蓝牙模块，提供ems设备启动操作，并可以提供参数配置、设备暂停等操作 |
| 蓝牙b | 与教练手机连接的蓝牙模块，在ems设备启动以后提供参数配置、设备暂停等操作 |

公共参数

参数 类型 是否必填 最大长度 描述
version string 是 3 调用的接口版本
device string 是 16 当前设备的id
method string 是 10 当前帧参数的含义
training_program int32 是 10 当前训练方案
training_time int32 是 2 当前训练时长
waveform string 否 2 当前刺激的波形
total_strength int32 否 3 当前刺激强度的最大值，从1-100
pulse_interval int32 否 2 指肌肉接受电流刺激时保持收缩状态的时间，单位为秒
pulse_pause int32 否 2 电流停止的时间，到下一次刺激的停顿时间，单位为秒
pulse_increment int32 否 2 脉冲提升间隔指脉冲频率提升至最高配置速率的时间
pulse_width int32 否 4 单次脉冲的持续时间决定了脉冲宽度
pulse_frequency int32 否 4 电流运动时每秒作用于肌群的脉冲次数
muscles list 否 可变 请求具体肌肉群的详细参数

version

当前空中接口定义为1.0版本，对应“1.0”。

device

ems设备的唯一id，16个字符表示，示例：
“1122334455667788”或者”AABBCCDDEEFF1100”。

method

定义当前帧的控制类型
| 字符串 | 含义 |
| — | — |
| start | 设备启动指令，只能蓝牙a通道响应 |
| pause | 设备暂停指令，用于特殊情况暂停设备运行 |
| continue | 设备继续指令，用于解除特殊情况后恢复设备运行 |
| stop | 设备停止指令，用于提前停止设备运行，单次训练完成，只能蓝牙a通道响应 |
| set | 设备设置指令，需要配合后面的其他参数使用 |
| get | 设备获取指令，获取当前设备运行状态中所有参数 |
| ack | 设备应答指令，针对上面所有指令的应答 |

training_program

该参数指定训练方案，有以下几种可选方案：
| 数值 | 含义 |
| — | — |
| 1 | 按摩 |
| 2 | 有氧 |
| 3 | 力量 |
| 4 | 自定义 |

training_time

该参数指定了训练时长，只对一次训练有效，指定了从设备开始到设备停止的时间。

waveform

该参数指示电极上的最终波形。
| 数值 | 含义 |
| — | — |
| 1 | 方波 |
| 2 | 矩形波 |
| 3 | 正弦波 |

pulse_interval

肌肉接受电流刺激时保持收缩状态的时间，一次刺激的持续时间，测量单位为秒

  • 超过100hz:1-2秒脉冲间隔
  • 70-100hz:2-5秒脉冲间隔
  • 70hz以下:5-10秒脉冲间隔
  • 20hz以下:直流

pulse_pause

电流停止的时间，到下一次刺激的停顿时间，最佳脉冲停顿比为1:1或1:1.5。最低停顿时间为200毫秒，否则将无法保证肌纤维的恢复。

pulse_increment

脉冲提升间隔指脉冲频率提升至最高配置速率的时间。针对方波有效

pulse_width

单次脉冲的持续时间决定了脉冲宽度。通常采用的脉冲宽度范围约为150-350ps。

pulse_frequency

电流运动时每秒作用于肌群的脉冲次数。

  • 20-50hz:I型肌纤维
  • 50-90hz:II型肌纤维，力量训练推荐80-85hz
  • 90-120hz:IIB型肌纤维

muscles

该list指定了实际10个肌肉群电极的单独刺激方案：
| 参数 | 类型 | 是否必填 | 最大长度 | 描述 |
| — | — | — | — | — | — |
| muscle | string | 是 | 10 | 指定配置的肌肉群 |
| strength | int32 | 是 | 3 | 指定肌肉群的刺激强度，0-100 |

  • muscle
    配置参数作用的指定肌肉群
    | 数值 | 含义 |
    | — | — |
    | 1 | 二头肌 |
    | 2 | 斜方肌|
    | 3 | 胸部|
    | 4 | 背部|
    | 5 | 腹部|
    | 6 | 腰部|
    | 7 | 四方肌|
    | 8 | 臀肌|
    | 9 | 额外|
    | 10 | 大腿肌肉的后部|
  • strength
    电压最大值（按摩、有氧训练方案下只能调节该值，其余5个参数无法配置）

基本配置

热身、动态恢复 1-10 持续 / 350

放松、按摩 100 1 1 150 5-10

新手配置

新手用户开始训练时总电平应设置为60至65%，在5到10次脉冲后、或肌肉已适应压力或客户发现强度不足时，逐渐增加负荷。
每节课程必须从对四肢施加零压力开始。

减脂肪

  1. 最大强度20秒
  2. 暂停10秒
  3. 重复8次
