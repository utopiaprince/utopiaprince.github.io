视频教程

  • 上传
    员工上传教练视频
  • 删除
    管理员可以删除员工上传的教练视频
  • 查看
    会员可以观看视频，学习如何使用智能设备

用户短视频

  • 上传
    用户可以上传自己的短视频，暂定15s上限
  • 删除
    用户可以删除自己上传的短视频
  • 查看
    其他所有用户/员工可以观看短视频
  • 点赞
    用户可以对视频进行点赞

移动管理

  • 员工app
    员工专用的app，支持员工的所有功能
  • 会员app
    会员专用的app，支持会员的所有功能
  • 支付宝接入
    嵌入支付宝功能，用户通过支付宝付费方式开启操作智能设备

智能设备管理

  • 设备信息
    智能设备型号、版本号、类型
  • 设备管理
    设备添加、修改、删除；
  • 数据交互
    智能设备可以把用户的锻炼信息转换成卡路里消耗，获取心率等信息
  • 设备操作记录
    设备在什么时间开启、什么时间结束，有哪位会员操作，使用率，在哪里使用的
  • 位置信息
    设备当前位置

收益管理

  • 流水记录
    记录最近时间智能设备的支付记录
  • 日、周、月收入
    用图表的方式展示收入，直观展示营收
