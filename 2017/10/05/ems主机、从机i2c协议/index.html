ems主机、从机i2c协议

前提

该文档定义了ems智能设备的主mcu与从mcu之间的i2c通信协议。从mcu根据4根地址线的值来定义多个的i2c地址。可以从0x20～0x2f。

简称 定义
R 只读
W 只写
R/W 可读可写

i2c接口

i2c接口定义了主mcu与从mcu的底层通信协议。

单字节写

参考hmc5983的i2c底层接口。

单字节读

参考hmc5983的i2c底层接口。

连续写

参考hmc5983的i2c底层接口。

连续读

参考hmc5983的i2c底层接口。

寄存器

地址 寄存器 含义 访问
0x00 Version 版本寄存器 R
0x01 Ctrl 控制寄存器 R/W
0x02 Status 状态寄存器 R
0x03 Training Program 训练项目寄存器 R/W
0x04 Training Time 训练时长寄存器 R/W
0x05 Waveform 波形寄存器 R/W
0x06 Local Strength 强度寄存器 R/W
0x07 Pulse Interval 脉冲间隔寄存器 R/W
0x08 Pulse Pause 脉冲停顿寄存器 R/W
0x09 Pulse Increment 脉冲提升寄存器 R/W
0x0a Pulse Width 脉冲宽度寄存器 R/W
0x0b Pulse Frequency 脉冲频率寄存器 R/W
0x30 Write Fifo 数据缓冲区写入区域 W
0x31 Read Fifo 数据缓冲区读取区域 R

Version

















Bit 7 6 5 4 3 2 1 0
Name version
名称 含义
version 版本寄存器，当前从mcu的软件版本。

Ctrl





















Bit 7 6 5 4 3 2 1 0
Name rfu start pause continue stop
名称 含义
start 启动控制位，通过写1来启动脉冲输出，已经启动则忽略
pause 暂停控制位，写1暂停脉冲输出，已经暂停则忽略
continue 恢复控制位，写1恢复脉冲输出，只有设备暂停才有效，已经恢复则忽略
stop 停止控制位，写1停止设备运行，完成本次训练

Status





















Bit 7 6 5 4 3 2 1 0
Name rfu start pause continue stop
名称 含义
start 启动状态位，读有效
pause 暂停状态位，读有效
continue 恢复状态位，读有效
stop 停止状态位，读有效

Training Program


















Bit 7 6 5 4 3 2 1 0
Name rfu program
  • program，训练方案，具体含义如下：
数值 含义
1 按摩
2 有氧
3 力量
4 自定义

Training Time

















Bit 7 6 5 4 3 2 1 0
Name time
  • time，训练时长，以s为单位。

Waveform


















Bit 7 6 5 4 3 2 1 0
Name rfu waveform
  • waveform，电极输出的波形.
    | 数值 | 含义 |
    | — | — |
    | 1 | 方波 |
    | 2 | 矩形波 |
    | 3 | 正弦波 |

Local Strength

















Bit 7 6 5 4 3 2 1 0
Name strength
  • strength，从mcu控制的电极输出电压（百分比）。

Pulse Interval

















Bit 7 6 5 4 3 2 1 0
Name pulse interval
  • pulse interval，肌肉接受电流刺激时保持收缩状态的时间，刺激的持续时间。单位为秒。

Pulse Pause

















Bit 7 6 5 4 3 2 1 0
Name pulse pause
  • pulse pause，电流停止的时间，到下一次刺激的停顿时间，最佳脉冲停顿比为1:1或1:1.5。最低停顿时间为200毫秒，否则将无法保证肌纤维的恢复。单位为秒。

Pulse Increment

















Bit 7 6 5 4 3 2 1 0
Name pulse increment
  • pulse increment，脉冲提升间隔指脉冲频率提升至最高配置速率的时间。针对矩形波有效。

Pulse Width

















Bit 7 6 5 4 3 2 1 0
Name pulse width
  • pulse width，单次脉冲的持续时间决定了脉冲宽度。通常采用的脉冲宽度范围约为150-350ps。

Pulse Frequency

















Bit 7 6 5 4 3 2 1 0
Name pulse frequency
  • pulse frequency，电流运动时每秒作用于肌群的脉冲次数。
